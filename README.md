### Installation
Install locally:

    * MongoDB
    * Node.js

### Setup

    git clone https://github.com/oli37/sportAppMERN.git
    cd skiApp
    npm install
    //npm install babel-loader babel-core babel-preset-env webpack
    //npm install eslint
    
    //start MongoDB before npm start
    
    npm start
    
Server should start concurrently with React-App
